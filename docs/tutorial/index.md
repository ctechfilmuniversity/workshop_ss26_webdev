# Tutorial - A Portfolio Page

With this tutorial, we build a personal portfolio website from scratch using Next.js, React, TypeScript, Tailwind CSS v4, and MDX. 

The result is a six-page static site: a home page with an animated hero, a collapsible CV, a filterable projects listing with detail pages, a contact page, and a legal impressum.
